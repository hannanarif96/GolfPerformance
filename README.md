# GolfPerformance

A mobile application for tracking and analyzing golf player performance, built as part of an Android development project.

## Features

- Player listing and management
- Shot tracking and performance analytics
- Modular architecture using clean separation of UI, domain, and data layers
- API integration for fetching player and shot data
- Built with modern Android development practices (Kotlin, Coroutines, Flow, ViewModel)

## Tech Stack

- Kotlin
- Android Jetpack (ViewModel, LiveData/StateFlow)
- Retrofit (API calls)
- Coroutines & Flow
- MVVM Architecture
- Unit Testing (JUnit, Mockito)
